# Dev Town Homebrew Tap

Homebrew formulas for Dev Town tools.

## Harbr

Install Harbr with:

```sh
brew install dev-town/tap/harbr
```

Or tap first:

```sh
brew tap dev-town/tap
brew install harbr
```

Harbr release artifacts are published from:

```text
https://github.com/dev-town/harbr/releases
```
